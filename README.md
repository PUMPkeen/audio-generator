# Audio Generator

Генератор аудио на базе модели ElevenLabs-v3 через Poe API.

## Установка
npm install
1. Установите зависимости:
```bash
npm install
```

2. Создайте файл `.env` на основе `.env.example`:
```bash
copy .env.example .env
```

3. Добавьте ваш Poe API ключ в файл `.env`:
```
POE_API_KEY=ваш_api_ключ_здесь
```

## Запуск

Запустите локальный сервер разработки:
```bash
npm run dev
```

Или:
```bash
npm start
```

Сервер запустится на `http://localhost:3000`

Откройте браузер и перейдите по адресу `http://localhost:3000`

## Использование

1. Введите текст в поле ввода
2. Нажмите кнопку "Сгенерировать аудио"
3. Дождитесь завершения генерации
4. Воспроизведите сгенерированное аудио

## Примечания

- Для работы приложения необходим валидный Poe API ключ
- Убедитесь, что ваш Poe API ключ имеет доступ к модели ElevenLabs-v3
